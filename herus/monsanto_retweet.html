2. What about Monsanto?¶

Let's now try to filter out all tweets/retweets not speaking about 'Monsanto'

To achieve this, we have two solutions :

  1. Join tables in an sql query and filter on documents.body
  2. Load the two tables and join afterwards

We first opt for the 2nd solution, so that for each document we will then have what users retweeted the tweet.

Solution 1¶

Query all the retweets of documents containing monsanto and select the doc as well.

From each dataset, we can extract a different number of retweets :

data20171 : 49648 
data20172 : 1666 
data20181 : 931 
data20182 : 558

Hence, it's only worth to keep retweets from the first dataset...
